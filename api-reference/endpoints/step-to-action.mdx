---
title: '/step_to_action'
description: 'Convert a step description into actionable browser commands'
api: 'POST /step_to_action'
---

This endpoint converts a natural language step description into structured browser interaction commands. The response is in natural language.

The interactions can be executed by the [browser actor](/api-reference/browser/browser-interactions) if desired.

### Request

<ParamField body="step" type="string" required>
  Natural language description of the step to take (e.g. "Click the login button")
</ParamField>

<ParamField body="api_key" type="string" required>
  Your API key for authentication
</ParamField>

### Response

Returns a string containing the browser interaction commands needed to complete the step.

### Example Request

```bash
curl -X POST 'https://api.simplex.sh/step_to_action' \
  -H "Content-Type: multipart/form-data" \
  -F 'step=Click the submit button' \
  -F 'api_key=your_api_key'
```

### Example Response

```json
"CLICK, the submit button"
```
