---

To use the Simplex API and SDK, you'll need an API key. Follow these simple steps to get started:

1. Visit the [Simplex Playground](https://www.simplex.sh/playground#api-request-form)
2. Fill out the API request form
3. Once approved, you'll receive your API key

## Using Your API Key

After receiving your API key, you can use it to authenticate your requests to the Simplex API.
