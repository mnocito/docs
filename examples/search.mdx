You can create function templates for common actions like searching.

```python search.py

def search(website, search_query):
    simplex.goto(website)
    simplex.click("the search bar")
    simplex.type(search_query)
    simplex.press_enter()

```


## Usage

```python
search("https://www.mit.edu/", "alyssa p hacker")
search("https://www.google.com/", "NVIDIA stock")
```
