All of these functions are avaliable on the `Simplex` class.
## goto
Navigate to a specific URL.

```python
def goto(self, url: str) -> None
```
<ParamField path="url" type="string" required>
  The URL to navigate to.
</ParamField>
### Example Usage

```python
from simplex import Simplex
simplex = Simplex(api_key="your_api_key")
simplex.goto("https://www.google.com")
```

## click
Click on an element on the page using a natural language description.

```python
def click(self, element_description: str, annotate: bool = True) -> None
```
<ParamField path="element_description" type="string" required>
  Natural language description of the element to click.
</ParamField>
<ParamField path="annotate" type="boolean" default="True">
  Whether to annotate the element on the browser.
</ParamField>

### Example Usage
```python
from simplex import Simplex
simplex = Simplex(api_key="your_api_key")
simplex.goto("https://www.google.com")
simplex.click("the search bar")
```

## scroll

```python
def scroll(scroll_amount: int) -> None
```
<ParamField path="scroll_amount" type="int" required>
  The amount of pixels to scroll by.
</ParamField>

### Example Usage
```python

simplex = Simplex(api_key="your_api_key")
simplex.goto("https://www.github.com/bensmidt")

```
