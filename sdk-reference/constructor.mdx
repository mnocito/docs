## Constructor

```python
client = Simplex(
    api_key: str,
    browser: Optional[Browser] = None
)
```

<ParamField path="api_key" type="string" required>
  API key for authentication.
</ParamField>

<ParamField path="browser" type="playwright.sync_api.Browser">
  Playwright browser object. If not provided, a new local browser instance will be created.
</ParamField>

## Sample Usage
```python
from simplex import Simplex
from playwright.sync_api import sync_playwright
with sync_playwright() as p:
        browser = p.chromium.connect_over_cdp("ws://remote-session-url")
        simplex = Simplex(api_key="your_api_key", browser=browser)
        # peform actions...
```
